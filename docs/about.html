🤖

多 Agent 协作

interview-answerer(答题) + quality-reviewer(审查) + doc-assembler(组装)。各司其职，通过明确定义的契约协作，而非一个巨型 prompt。

✅

质量门控

15 项清单逐题审查，不合格自动重答（最多2次），不静默丢弃。确保每道题的输出都达到面试可用标准。

📚

题库沉淀

每道题自动分类+自动标签+MD5去重+写独立.md文件。13个分类目录，index.json索引。数据永久保存、可检索。

🚀

自动部署

答题→审查→归档→MD→HTML→data.json→git push→GitHub Pages，全自动10阶段流水线，无需手动操作。

📅

每日复习

复习打卡、连续天数追踪、30天月历、进度条、一键全标记。localStorage持久化，隐私安全。

🐍

Python 独立运行

9 个 Python 脚本覆盖全流程，不依赖 Claude Code。Anthropic API Key 即可独立答题、归档、训练。
