# MySQL有哪几种引擎？

主要引擎对比：InnoDB（支持事务/行锁/外键/MVCC/崩溃恢复）、MyISAM（不支持事务/表锁/全文索引）。其他：Memory（内存表）、CSV、Archive（高压缩比）。
