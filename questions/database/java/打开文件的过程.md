# 打开文件的过程

从Java代码到磁盘的完整链路：Java native方法→glibc fopen→系统调用sys_open→VFS查找dentry和inode→分配文件描述符→检查page cache→DMA从磁盘读取→数据拷贝到用户空间。关键结构：task_struct/file_struct/struct file/inode/dentry。
