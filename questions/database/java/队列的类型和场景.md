# 队列的类型和场景

FIFO普通队列（任务队列）、优先队列（紧急任务/Dijkstra）、延迟队列（订单超时取消）、阻塞队列（生产者-消费者）、双端队列Deque（滑动窗口/LRU）、循环队列（有界缓冲）。MQ产品：Kafka（高吞吐）、RocketMQ（事务消息）、RabbitMQ（灵活路由）。
