# 数据库的隔离级别有哪几种？

四种隔离级别：READ UNCOMMITTED（脏读）、READ COMMITTED（不可重复读）、REPEATABLE READ（可重复读）、SERIALIZABLE（串行化）。MySQL默认RR，Oracle默认RC。RR通过Next-Key Lock解决幻读。
