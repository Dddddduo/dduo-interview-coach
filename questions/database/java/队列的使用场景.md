# 队列的使用场景

三大场景：异步解耦（订单→MQ→多系统消费）、削峰填谷（秒杀百万请求→队列缓冲→后端逐步处理）、任务调度（延迟消息、定时任务）。具体应用：日志收集ELK、异步通知、分布式事务(MQ最终一致性)。
