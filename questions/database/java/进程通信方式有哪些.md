# 进程通信方式有哪些？

6种IPC：管道（Pipe单向流）、消息队列（异步通信）、共享内存（最快）、信号量（同步互斥）、信号（异步通知）、套接字（跨网络）。共享内存最快（零拷贝），需配合信号量同步。
