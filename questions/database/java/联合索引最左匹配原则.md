# 联合索引最左匹配原则

联合索引idx_a_b_c(a,b,c)必须从最左列开始匹配。WHERE a=1 AND c=3只能a走索引（b被跳过）。WHERE b=2不走索引（没从a开始）。范围查询后的列无法使用索引：WHERE a>1 AND b=2只a走索引。
