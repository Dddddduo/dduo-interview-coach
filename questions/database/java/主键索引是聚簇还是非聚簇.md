# 主键索引是聚簇还是非聚簇？

InnoDB中主键索引=聚簇索引（Clustered Index）。数据行直接存在B+树叶子节点。每张表只有一个聚簇索引。无主键时选唯一非空索引，都没有就隐式生成ROWID。
