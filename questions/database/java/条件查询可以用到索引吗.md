# 条件查询可以用到索引吗？

能用索引：等值(=)、IN、范围(>/</BETWEEN)、前缀LIKE'abc%'、IS NULL。可能不能用：左模糊、函数包列、隐式类型转换、OR配非索引列。取决于WHERE条件的写法是否触发失效规则。
