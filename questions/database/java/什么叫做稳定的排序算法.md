# 什么叫做稳定的排序算法

稳定排序（Stable Sort）：排序前后相等元素的相对次序保持不变。重要用于多关键字排序和基数排序基础。稳定：冒泡、插入、归并；不稳定：选择、快排、堆排。判断关键是相等元素是否交换位置。
