# 进程有几个状态？怎么切换？

三态模型：运行态、就绪态、阻塞态。五态增加创建态和终止态。切换：就绪→运行（调度），运行→就绪（时间片用完），运行→阻塞（等待I/O/资源），阻塞→就绪（I/O完成）。Linux：TASK_RUNNING/INTERRUPTIBLE/UNINTERRUPTIBLE/STOPPED/ZOMBIE。
