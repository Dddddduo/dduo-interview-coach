# 选择排序了解吗？

选择排序（Selection Sort）每轮从未排序区间选出最小值放到已排序区间末尾。优点：实现简单、数据移动次数最少（O(n)次交换）。缺点：无论数据是否有序都需要O(n²)次比较、不稳定。
