# 快速选择的时间复杂度

快速选择（Quickselect）平均时间O(n)，最坏O(n²)。每次partition只递归一边，T(n)=T(n/2)+O(n)→O(n)。最坏情况pivot选到极端值。空间O(1)原地操作。
