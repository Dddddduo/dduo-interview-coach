# MySQL索引结构是怎样的？

B+树索引：多路平衡树，非叶子节点只存索引键（高扇出），叶子节点存完整数据（聚簇索引）或主键值（二级索引），叶子节点双向链表连接。3-4层支持千万级数据。优势：磁盘IO少、范围查询高效。
