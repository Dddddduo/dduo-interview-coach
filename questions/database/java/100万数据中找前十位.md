# 100万数据中找前十位

Top K问题用最小堆最优。维护大小为10的最小堆，遍历数据时与堆顶比较，大于堆顶则替换并调整。时间O(nlogK)≈O(n)，空间O(K)=O(1)，支持流式数据。对比：全排O(nlogn)不必要，快速选择无法处理流式数据。
