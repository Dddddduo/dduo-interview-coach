# 进程和线程有什么区别？

进程是资源分配最小单位（独立地址空间），线程是CPU调度最小单位（共享进程资源）。切换开销：进程大（切换页表、刷新TLB），线程小。独立性：一个进程崩溃不影响其他，线程崩溃可能导致整个进程退出。通信：进程IPC，线程共享变量。
