# 手撕代码：母牛生小牛问题

### 🧠 联想记忆法

**记忆口诀**："今年的牛 = 去年的牛 + 三年前出生的牛"

**记忆原理**：
- 递推关系 f(n) = f(n-1) + f(n-3) 的核心逻辑：第 n 年的牛由两部分组成——所有去年就在的牛（没死），以及今年新出生的小牛
- 新出生的小牛数量 = 第 n-3 年时存在的母牛数量，因为每头母牛从出生后第 4 年开始每年生一头，意味着在第 n-3 年存在的牛，到第 n 年时刚好满 3 岁（进入第 4 年），具备生育能力
- 这个递推与斐波那契数列（Fibonacci sequence）同构，只是间隔从 2 变成了 3

**关联知识**：
- **斐波那契数列**（Fibonacci sequence）：F(n) = F(n-1) + F(n-2)
- **跳台阶问题**（Climbing Stairs）：本质也是线性递推
- **泰波那契数列**（Tribonacci sequence）：T(n) = T(n-1) + T(n-2) + T(n-3)
- **矩阵快速幂**（Matrix Fast Exponentiation）：所有线性齐次递推的通解方法
- **动态规划**（Dynamic Programming）：递推关系的最优子结构

### 📖 深度解答

#### 一、核心概念：问题分析与递推公式推导

定义 f(n) 为第 n 年年初的母牛总数。递推关系推导如下：

第 n 年的牛 = 第 n-1 年的牛（全部存活）+ 第 n 年新出生的小牛

难点在于计算"第 n 年新出生的小牛数量"。每头母牛从出生后的第 4 年开始每年年初生一头小母牛，这意味着在第 n 年年初，所有年龄 >= 3 岁的牛（即出生年份 <= n-3 的牛）都会生一头小牛。第 n-3 年年初存在的每头牛，到第 n 年年初时都已经活了至少 3 年（进入第 4 年），所以它们都有生育能力。因此：第 n 年新生牛数量 = f(n-3)

于是得到递推公式：f(n) = f(n-1) + f(n-3)

**边界条件**（Base Case）：
- f(1) = 1：第1年只有最初的母牛（刚出生）
- f(2) = 1：第2年小牛才1岁，不能生育
- f(3) = 1：第3年小牛2岁，不能生育
- f(4) = 2：f(4) = f(3) + f(1) = 1 + 1 = 2

**手动验证前几年**：
| 年份 | 计算过程 | f(n) | 说明 |
|------|---------|------|------|
| 1 | - | 1 | 初始母牛 |
| 2 | - | 1 | 无新生 |
| 3 | - | 1 | 无新生 |
| 4 | f(3)+f(1)=1+1 | 2 | 初始母牛生1头 |
| 5 | f(4)+f(2)=2+1 | 3 | 初始母牛再生 |
| 6 | f(5)+f(3)=3+1 | 4 | 初始母牛再生 |
| 7 | f(6)+f(4)=4+2 | 6 | 第一头小牛开始生育 |
| 8 | f(7)+f(5)=6+3 | 9 | 第二头小牛开始生育 |

#### 二、底层原理：三种解法及完整 Java 实现

##### 解法一：递归 + 备忘录（Top-Down Memoization）

```java
import java.util.HashMap;
import java.util.Map;

public class CowProblemMemo {

    private static Map<Integer, Long> memo = new HashMap<>();

    public static long countCows(int n) {
        if (n < 1) {
            throw new IllegalArgumentException("n must be >= 1, got: " + n);
        }
        if (n <= 3) {
            return 1;
        }
        if (memo.containsKey(n)) {
            return memo.get(n);
        }
        long result = countCows(n - 1) + countCows(n - 3);
        memo.put(n, result);
        return result;
    }

    public static void main(String[] args) {
        int[] testYears = {1, 2, 3, 4, 5, 6, 7, 8, 10, 20, 30, 50};
        for (int year : testYears) {
            memo.clear();
            System.out.printf("f(%d) = %d%n", year, countCows(year));
        }
    }
}
```

**时间复杂度**（Time Complexity）：O(n)，每个 n 只计算一次
**空间复杂度**（Space Complexity）：O(n)，递归栈深度 + 备忘录存储

##### 解法二：动态规划迭代（Bottom-Up DP, O(n) Time, O(1) Space）

```java
public class CowProblemDP {

    public static long countCows(int n) {
        if (n < 1) {
            throw new IllegalArgumentException("n must be >= 1, got: " + n);
        }
        if (n <= 3) {
            return 1;
        }
        long fn3 = 1; // f(1)
        long fn2 = 1; // f(2)
        long fn1 = 1; // f(3)
        long fn = 0;

        for (int i = 4; i <= n; i++) {
            fn = fn1 + fn3;     // f(n) = f(n-1) + f(n-3)
            fn3 = fn2;          // 滚动
            fn2 = fn1;
            fn1 = fn;
        }
        return fn;
    }

    public static void main(String[] args) {
        int[] testYears = {1, 2, 3, 4, 5, 6, 7, 8, 10, 20, 30, 50, 100};
        for (int year : testYears) {
            System.out.printf("f(%d) = %d%n", year, countCows(year));
        }
    }
}
```

**时间复杂度**：O(n)，单层循环
**空间复杂度**：O(1)，仅使用 4 个局部变量

##### 解法三：矩阵快速幂（Matrix Fast Exponentiation, O(log n)）

递推公式 f(n) = f(n-1) + f(n-3) 的矩阵形式：

```
| f(n)   |     | 1  0  1 |     | f(n-1) |
| f(n-1) |  =  | 1  0  0 |  *  | f(n-2) |
| f(n-2) |     | 0  1  0 |     | f(n-3) |
```

记变换矩阵 M = [[1,0,1],[1,0,0],[0,1,0]]，则：
| f(n)   |       (n-3)    | f(3) |
| f(n-1) |  =  M         * | f(2) |
| f(n-2) |                 | f(1) |

通过快速幂（Binary Exponentiation）计算 M^(n-3) 可在 O(log n) 时间内得到结果。

```java
public class CowProblemMatrix {

    public static long[][] multiply(long[][] a, long[][] b) {
        int size = 3;
        long[][] result = new long[size][size];
        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                for (int k = 0; k < size; k++) {
                    result[i][j] += a[i][k] * b[k][j];
                }
            }
        }
        return result;
    }

    public static long[][] matrixPow(long[][] matrix, int power) {
        int size = 3;
        long[][] result = new long[size][size];
        for (int i = 0; i < size; i++) {
            result[i][i] = 1;  // 单位矩阵
        }
        long[][] base = matrix;
        int exp = power;
        while (exp > 0) {
            if ((exp & 1) == 1) {
                result = multiply(result, base);
            }
            base = multiply(base, base);
            exp >>= 1;
        }
        return result;
    }

    public static long countCows(int n) {
        if (n < 1) {
            throw new IllegalArgumentException("n must be >= 1, got: " + n);
        }
        if (n <= 3) {
            return 1;
        }
        long[][] M = {
            {1, 0, 1},
            {1, 0, 0},
            {0, 1, 0}
        };
        long[][] Mn = matrixPow(M, n - 3);
        return Mn[0][0] * 1 + Mn[0][1] * 1 + Mn[0][2] * 1;
    }

    public static void main(String[] args) {
        int[] testYears = {1, 2, 3, 4, 5, 6, 7, 8, 10, 20, 30, 50, 100, 1000};
        for (int year : testYears) {
            System.out.printf("f(%d) = %d%n", year, countCows(year));
        }
    }
}
```

**时间复杂度**：O(log n)，矩阵快速幂每次将幂次减半
**空间复杂度**：O(1)，仅使用常量大小的 3x3 矩阵

#### 四、深入思考

**1. 数值溢出问题**（Overflow Issue）：f(n) 增长非常快，近似于 O(α^n) 其中 α ≈ 1.46557（特征方程 x^3 - x^2 - 1 = 0 的实根）。当 n > 73 时，f(n) 超过 Java long 最大值（9.22e18）。对于更大的 n，应使用 BigInteger。

**2. 空间优化对比**：
| 解法 | 时间复杂度 | 空间复杂度 | 适用场景 |
|------|-----------|-----------|---------|
| 朴素递归 | O(α^n) | O(n) | 理论推导，不适用实际 |
| 递归+备忘录 | O(n) | O(n) | n < 10^6 且能承受栈深度 |
| DP迭代 | O(n) | O(1) | 工程首选，n < 10^7 |
| 矩阵快速幂 | O(log n) | O(1) | 极端大 n（n > 10^9） |

**3. 递推关系的推广**（Generalization）：若牛从第 k 年开始生育，递推公式为 f(n) = f(n-1) + f(n-k+1)。当 k=2 时为斐波那契数列，k=4 时增长率约为 1.380。

- 第二步（2分钟）：从递归到备忘录。先给出朴素递归，指出指数爆炸问题。引入备忘录优化到 O(n)。

#### 常见误区
1. 递推公式写成 f(n) = f(n-1) + f(n-4)：第 4 年开始 = 过了 3 年，所以是 +f(n-3)
2. 边界条件遗漏：导致递归栈溢出
3. 数据类型选择不当：f(73) 超过 long 上限
4. 备忘录作用域问题：多个测试用例共用不清空
5. 矩阵乘法维度错误：递推三阶，矩阵须 3x3
