# 除了加索引还有什么优化方案？

不加索引也能优化：数据归档（冷热分离）、硬件升级（SSD+大内存）、缓存（Redis+本地缓存）、SQL重写（优化JOIN/避免子查询）、读写分离、分库分表、连接池调优（HikariCP）。
