# HTTP和HTTPS有什么区别？

六大区别：HTTP明文vs HTTPS加密（SSL/TLS）；端口80 vs 443；HTTPS需CA证书；HTTPS握手多2-RTT；搜索引擎优先收录HTTPS；HTTPS防篡改防窃听防冒充。
