# 如何分表？按什么维度？

水平分表：按ID范围（user_0/user_1）、按哈希（hash(id)%16）、按时间（按月order_202401）、按地理。垂直分表：冷热分离（常用字段主表+大字段扩展表）、业务拆分（基本信息表+详情表）。
