# Dijkstra算法解决什么问题？

Dijkstra算法解决加权图中单源最短路径问题，要求边权非负。使用优先队列优化，时间O((V+E)logV)。应用：地图导航、OSPF路由协议、社交网络最短关系链。不能处理负权边（用Bellman-Ford）。
