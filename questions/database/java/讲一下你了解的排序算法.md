# 讲一下你了解的排序算法

三类排序算法：O(n²)（冒泡、选择、插入）适合小数据量；O(nlogn)（归并、快排、堆排）适合大数据量；O(n)（桶排、计数、基数）特殊场景。快排平均最快，归并稳定，堆排原地。选型建议：小数据插入排序，大数据快排，需稳定归并，Top K堆排。
