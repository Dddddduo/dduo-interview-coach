# 非对称加密和对称加密区别

对称加密（AES/DES）一把密钥同时加解密，速度快但有密钥分发问题。非对称加密（RSA/ECC）公钥加密私钥解密，密钥分发安全但速度慢（慢1000倍）。HTTPS混合使用：非对称交换对称密钥，对称加密传输数据。
