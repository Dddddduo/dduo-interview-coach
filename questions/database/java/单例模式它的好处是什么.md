# 单例模式，它的好处是什么？

单例模式（Singleton Pattern）确保一个类只有一个实例并提供全局访问点。好处：节省内存、统一控制、延迟加载、全局访问、线程安全、避免重复创建。实现方式：饿汉式、双检锁DCL、静态内部类、枚举。
