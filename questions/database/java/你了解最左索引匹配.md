# 你了解最左索引匹配

联合索引idx_a_b_c(a,b,c)遵循最左前缀原则：WHERE条件必须从最左列开始。举例：WHERE a=1走a，WHERE a=1 AND b=2走a,b，WHERE b=2不走索引（没从a开始）。跳过中间列会导致后续列无法使用索引。
