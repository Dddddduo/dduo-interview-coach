# 用户态线程和内核线程有什么区别？

内核线程由OS管理（抢占式调度），用户态线程由用户态线程库管理（协作式调度）。用户态线程创建极快（纳秒级）但无法利用多核，内核线程支持多核但创建较慢（需系统调用）。Java采用1:1模型。协程是轻量化用户态线程。
