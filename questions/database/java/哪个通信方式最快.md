# 哪个通信方式最快？

共享内存最快，数据不需要在内核和用户空间之间拷贝（零拷贝）。排名：共享内存（零拷贝）> 管道 > 消息队列 > UDS Socket > TCP Socket。共享内存需配合信号量同步。
