# 优先级调度算法讲一下

优先级调度每次选优先级最高的进程执行。静态优先级（创建时确定）和动态优先级（随时间调整）。问题：低优先级可能饥饿。解决方案：优先级老化（Aging）。Linux用140优先级队列+位图实现O(1)调度。
