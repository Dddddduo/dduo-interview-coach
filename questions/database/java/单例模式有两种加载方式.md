# 单例模式有两种加载方式

饿汉式（Eager Loading）：类加载时创建实例，天然线程安全；懒汉式（Lazy Loading）：首次调用getInstance()时创建，需处理线程安全。选择原则：实例一定会被使用且创建开销小选饿汉式，否则选懒汉式。
