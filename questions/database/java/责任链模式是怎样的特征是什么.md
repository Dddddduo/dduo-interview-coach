# 责任链模式是怎样的？特征是什么？

责任链模式（Chain of Responsibility Pattern）是行为型设计模式，将请求发送者和接收者解耦。多个处理对象连成链，请求沿链传递直到被处理。核心特征：链式结构、请求传递、解耦、动态组合。典型应用：Filter链、Interceptor链、Netty ChannelPipeline。
